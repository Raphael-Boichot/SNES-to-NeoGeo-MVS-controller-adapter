# SNES to NeoGeo MVS controller adapter
A SNES to NeoGeo MVS controller adapter based on Arduino. To be used with a Supergun, not AES compatible. The code and pinout come from [this repository](https://github.com/robinhedwards/SNES-to-NeoGeo), must works as it, but to be optimized.
